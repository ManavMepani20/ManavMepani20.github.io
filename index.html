Robotic Arm Control — ROS 2

WPI · Graduate Projects (RBE 500) · Oct–Dec 2024

A series of progressively complex ROS 2 control implementations for the OpenManipulator-X — from velocity-level kinematics and linear path motion through to a from-scratch PD controller running at 100 Hz in current control mode.

  • Built 3 Python ROS2 nodes: vel_kinematics (Jacobian & twist computation), vel_calc (incremental joint positions), vel_pub (constant twist publisher) for linear Cartesian path execution
  • Implemented Jacobian matrix (linear velocities via cross-product, angular from joint orientations) and Moore-Penrose pseudoinverse for joint-velocity resolution
  • Developed PD controller for Actuator 4 in current control mode (DYNAMIXEL X series, 1 MHz baud); control law: effort = Kp·error − Kd·derivative
  • Tuned gains to Kp = 0.19, Kd = 0.03 — achieving fast convergence with minimal overshoot across 3 reference tests: 930→2000, 2180→1500, and 930→1800 encoder units
  • Logged 10 seconds of position data at 0.01 s sampling time (100 Hz); results visualized with Python/matplotlib
  • Validated FK by echoing /end_effector_pose across 3 distinct joint configurations using ROS 2 service calls

Autonomous Cooking System — Undergrad Thesis

The system integrates a carousel solid-ingredient dispenser, liquid/batter dispensers, and a three-fingered claw end-effector.

  • Optimized link lengths via Simulated Annealing to minimize peak joint torque; shoulder torque = 616.89 Ncm, elbow = 357.22 Ncm, wrist = 239.93 Ncm at full horizontal extension
  • Performed ANSYS static structural FEA on aluminum links (1 mm wall) and transient thermal analysis — validated experimentally within ~8% error across 30–125°C hot-plate range
  • Simulated IK using geometric kinematic decoupling in Webots (Python); interpolated joint-space trajectories with C²-continuous sin² velocity profiles
  • Controlled physical prototype via Arduino Mega with 10-turn potentiometers (50V supply) achieving 0.4° sensor resolution; error corrected to ±1° via linear regression calibration
  • Payload capacity tested at ≥42.7 kg-cm at the first joint; 5 of 6 DOF fully operational on physical prototype
  • Published in Materials Today: Proceedings, ICAME22 Conference (2022) — Vol. 68, Part 6, pp. 1930–1945

Pulley-Driven Crimping Tool Attachment

Machine Design Course Project · SPCE

Designed a motorized crimping tool attachment to eliminate the large manual forces that cause repetitive strain injury (carpal tunnel syndrome) in manufacturing workers. The device uses a 5-pulley rope drive system actuated by a DC motor and Li-Po battery — achieving crimp forces that would otherwise require dangerous manual effort.

  • 5-pulley rope drive provides a mechanical advantage of 6, designed for a max crimping load of 1000 N (actual peak load 645 N + 5.1 N self-weight + 27.55 N spring force = 678.76 N)
  • Motor selection: Rhino IG32 12V planetary DC motor, 686.7 Ncm torque at 30 RPM — completes a full crimp cycle in 3.9 seconds (within 10 s client requirement)
  • Wire rope: 6×19 fiber core, 3 mm diameter, D/dr = 23, static FOS = 5.48, rated for 90,000 bends to fatigue failure (≈7,500 crimping cycles)
  • Rope drum and sheaves designed per IS 3177; combined stress verified below allowable stress of 152.38 N/mm² for FG 200 cast iron
  • SKF 608 deep-groove ball bearing selected; relubrication interval 12,300 hrs; frictional torque per bearing 9.63 Nmm
  • 4-bar parallelogram spring return mechanism (k = 0.1 N/mm, max spring force 27.44 N) provides passive reset — no motor effort required to open tool
  • Device dimensions within 300×100×150 mm; total weight <5 kg; powered by 12V Li-Po battery with 3-position DPDT switch for forward/off/reverse

Two-Stage Reverted Helical Gearbox

Machine Design Course Project · SPCE

Designed a complete two-stage reverted helical gearbox to deliver 49 kW at 95% efficiency, reducing speed from 1000 RPM to 179 RPM (gear ratio 5.59:1). All components — gears, shafts, bearings, keys, and lubrication — were designed from first principles against a full set of client specifications.

  • Gear parameters: 19-45 tooth combination (optimized via Python iteration for coprime teeth, minimal center distance, and equal stage ratios), module mn = 8 mm, helix angle 25°, 20° pressure angle, 0.55% Carbon Chromium Steel (UTS 850 MPa, 500 BHN)
  • Gear life: 17,833 hrs for the critical Gear 3 (pitting resistance), with Gear 1 exceeding 5.11×10⁶ hrs; all verified using Shigley's YN/ZN stress cycle factors at 95% reliability
  • Shaft design (ASME code, AISI 1040 steel): Input ø50/30 mm hollow (L=178 mm), Intermediate ø80/48 mm (L=402 mm), Output ø90/54 mm (L=194 mm) — all passed angle-of-twist, deflection, and slope constraints
  • Bearings (SKF deep-groove ball): 6210-2Z (input), 6216-2Z (intermediate), 6218-2Z (output) — SKF rating lives of 14,600 / 21,500 / 12,500 hrs respectively; pre-greased with MT33 (grease life up to 20,600 hrs)
  • Gear lubrication: Shell Omala S2 G 320 (ISO VG 320), splash method, calculated lubricant life 9,000 hrs (exceeds 8,000 hr change interval); lubrication check every 2,000 hrs
  • Total gearbox weight: 329.3 kg; noise target <85 dB at 1 m; IoT integration proposed for remote oil temperature, viscosity, RPM, and acoustic emission monitoring
