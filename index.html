6-DOF IK Solver Benchmarking for VR Teleoperation

WPI · Graduate Project (RBE 500) · Oct–Dec 2024

Implemented and benchmarked four IK solvers for the OpenManipulator-X robot using standard Denavit-Hartenberg (DH) parameters and ROS 2. Built a full VR-to-robot teleoperation pipeline for real-time manipulation.

  • Implemented Newton-Raphson (NR), I-DLS, SVD-DLS, and Particle Swarm Optimization (PSO) solvers
  • FK node subscribes to /joint_states, computes end-effector pose via DH conventions, and publishes to /end_effector_pose
  • IK service (/calculate_ik) accepts desired end-effector pose and returns joint angles using quaternion-based orientation handling with error checks
  • Validated across 3 distinct joint configurations; bonus pick-and-place task completed with gripper control via /goal_tool_control
  • Developed a VR-to-robot pipeline using Unity, ROS, and MATLAB for real-time teleoperation
  • Built a performance evaluation framework analyzing computation time, convergence, and joint-space efficiency

Robotic Arm Control — ROS 2

WPI · Graduate Projects (RBE 500) · Oct–Dec 2024

A series of progressively complex ROS 2 control implementations for the OpenManipulator-X — from velocity-level kinematics and linear path motion through to a from-scratch PD controller running at 100 Hz in current control mode.

  • Built 3 Python ROS2 nodes: vel_kinematics (Jacobian & twist computation), vel_calc (incremental joint positions), vel_pub (constant twist publisher) for linear Cartesian path execution
  • Implemented Jacobian matrix (linear velocities via cross-product, angular from joint orientations) and Moore-Penrose pseudoinverse for joint-velocity resolution
  • Developed PD controller for Actuator 4 in current control mode (DYNAMIXEL X series, 1 MHz baud); control law: effort = Kp·error − Kd·derivative
  • Tuned gains to Kp = 0.19, Kd = 0.03 — fast convergence with minimal overshoot across 3 reference tests: 930→2000, 2180→1500, and 930→1800 encoder units
  • Logged 10 seconds of data at 100 Hz; visualized with Python/matplotlib

Autonomous Cooking System — Undergrad Thesis

SPCE · Undergrad Thesis · Aug 2021–May 2022

Designed, fabricated, and validated a 6-DOF PUMA-configured robotic arm for autonomous culinary assistance — targeting three Indian recipes (Bhindi Masala, Pav Bhaji, Dosa). The system integrates a carousel solid-ingredient dispenser, liquid/batter dispensers, and a three-fingered claw end-effector.

  • Optimized link lengths via Simulated Annealing; shoulder torque = 616.89 Ncm, elbow = 357.22 Ncm, wrist = 239.93 Ncm at full horizontal extension
  • ANSYS static structural FEA on aluminum links (1 mm wall) and transient thermal analysis — validated experimentally within ~8% error across 30–125°C
  • Simulated IK using geometric kinematic decoupling in Webots (Python); C²-continuous sin² velocity profiles for smooth joint-space trajectories
  • Arduino Mega control with 10-turn potentiometers; 0.4° sensor resolution, error corrected to ±1° via linear regression calibration
  • Payload capacity ≥42.7 kg-cm at the first joint; 5 of 6 DOF fully operational on physical prototype
  • Published in Materials Today: Proceedings, ICAME22 (2022) — Vol. 68, Part 6, pp. 1930–1945

Pulley-Driven Crimping Tool Attachment

Machine Design Course Project · SPCE

Designed a motorized crimping tool attachment to eliminate the large manual forces that cause repetitive strain injury in manufacturing workers. The device uses a 5-pulley rope drive actuated by a DC motor and Li-Po battery.

  • 5-pulley rope drive provides mechanical advantage of 6; max crimping load 1000 N (actual peak 678.76 N including self-weight and spring force)
  • Motor: Rhino IG32 12V, 686.7 Ncm at 30 RPM — full crimp cycle in 3.9 s (within 10 s requirement)
  • Wire rope: 6×19 fiber core, 3 mm dia, D/dr = 23, static FOS = 5.48, fatigue life 90,000 bends (≈7,500 crimp cycles)
  • SKF 608 bearing, relubrication interval 12,300 hrs; 4-bar spring return (k = 0.1 N/mm) for passive open reset
  • Envelope: 300×100×150 mm; weight <5 kg; 12V Li-Po with DPDT forward/off/reverse switch

Two-Stage Reverted Helical Gearbox

Machine Design Course Project · SPCE

Designed a complete two-stage reverted helical gearbox delivering 49 kW at 95% efficiency, reducing speed from 1000 RPM to 179 RPM (ratio 5.59:1). All components designed from first principles against a full set of client specifications.

  • Gears: 19-45 tooth combination (Python-optimized), mn = 8 mm, helix 25°, 20° pressure angle; 0.55% Carbon Chromium Steel, UTS 850 MPa
  • Gear life: 17,833 hrs (critical Gear 3, pitting), Gear 1 >5.11×10⁶ hrs; verified via Shigley's YN/ZN factors at 95% reliability
  • Shafts (AISI 1040): Input ø50/30 mm hollow (L=178 mm), Intermediate ø80/48 mm (L=402 mm), Output ø90/54 mm (L=194 mm)
  • SKF bearings: 6210-2Z / 6216-2Z / 6218-2Z — lives 14,600 / 21,500 / 12,500 hrs; MT33 grease (life up to 20,600 hrs)
  • Lubrication: Shell Omala S2 G 320, calculated life 9,000 hrs; total weight 329.3 kg; noise <85 dB at 1 m
